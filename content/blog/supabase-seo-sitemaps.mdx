## Sitemap Generation

Automated sitemap generation for all properties:

```typescript title="Sitemap generation"
async function generateSitemap() {
  const pages = await getAllPages()
  const sitemap = pages.map(page => ({
    url: `https://supabase.com${page.path}`,
    lastmod: page.updatedAt,
    priority: page.priority,
  }))
  return toXML(sitemap)
}
```

## Meta Tags

Comprehensive meta tag coverage:

```tsx title="Page metadata"
<Head>
  <title>{page.title} | Supabase</title>
  <meta name="description" content={page.description} />
  <meta property="og:image" content={page.ogImage} />
  <meta name="twitter:card" content="summary_large_image" />
</Head>
```

## Video Meta Tags

For pages with video content:

```tsx title="Video meta tags"
<meta property="og:video" content={videoUrl} />
<meta property="og:video:type" content="video/mp4" />
<meta property="og:video:width" content="1280" />
```

Code editing in the browser needs to feel native. I integrated Monaco editor across the Supabase dashboard for SQL editing and data viewing.

## SQL Editor

The SQL editor is where developers spend significant time:

- Writing and testing queries
- Creating migrations
- Debugging data issues
- Exploring schemas

### Dark Theme

Monaco needed to match the Supabase aesthetic:

```typescript title="Custom Monaco theme"
monaco.editor.defineTheme('supabase-dark', {
  base: 'vs-dark',
  inherit: true,
  rules: [
    { token: 'keyword', foreground: '3ECF8E' },
    { token: 'string', foreground: 'F472B6' },
  ],
  colors: {
    'editor.background': '#1a1a1a',
  },
})
```

### SQL Persistence

I built support for saving SQL content:

- Queries persist across sessions
- Named snippets for reuse
- Folder organization
- Share functionality

## JSON Panel

For viewing JSON data in the Table Editor:

```tsx title="JSON panel component"
<MonacoEditor
  language="json"
  value={JSON.stringify(row.data, null, 2)}
  options={{
    readOnly: true,
    minimap: { enabled: false },
    scrollBeyondLastLine: false,
  }}
/>
```

This makes complex JSON fields readable.

## Performance Fixes

### Lazy Loading

Monaco is heavy. I implemented lazy loading:

```typescript title="Dynamic import"
const MonacoEditor = dynamic(
  () => import('@monaco-editor/react'),
  { ssr: false, loading: () => <Skeleton /> }
)
```

### MobX Integration

Fixed issues with MobX reactions and SQL state:

- SQL editor no longer runs "previous" version of changes
- Proper state synchronization
- Debounced saves to prevent spam

## Selection Issues

I fixed selection issues in Monaco:

- Text selection works properly
- Copy/paste functions correctly
- Cursor positioning accurate
- Keyboard shortcuts work

## AI Integration

Monaco powers the AI SQL editor:

- Prompt input with syntax highlighting
- Generated SQL display
- Diff view for suggestions
- Brand color integration

## Impact

Monaco integration provides:

1. Familiar editing experience for developers
2. Syntax highlighting for SQL and JSON
3. Keyboard shortcuts (Ctrl+Enter to run)
4. Auto-completion for SQL

A good editor makes the database accessible.

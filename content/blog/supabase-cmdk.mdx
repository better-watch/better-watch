Command palettes have become essential for power usersâ€”a quick way to navigate, search, and execute actions without reaching for the mouse. I implemented CMDK (Command K) for Supabase, bringing this pattern to the dashboard and documentation.

## The Implementation

The command palette appears on `Cmd/Ctrl + K` and provides:

- **Quick navigation** to any page in the dashboard
- **Search** across projects, tables, and documentation
- **Actions** like creating new projects or tables
- **Recent items** for fast access

## Technical Stack

The implementation uses the `cmdk` library with custom styling:

```tsx title="CMDK setup"
import { Command } from 'cmdk'

function CommandPalette() {
  const [open, setOpen] = useState(false)
  
  useEffect(() => {
    const down = (e: KeyboardEvent) => {
      if (e.key === 'k' && (e.metaKey || e.ctrlKey)) {
        e.preventDefault()
        setOpen((open) => !open)
      }
    }
    
    document.addEventListener('keydown', down)
    return () => document.removeEventListener('keydown', down)
  }, [])
  
  return (
    <Command.Dialog open={open} onOpenChange={setOpen}>
      <Command.Input placeholder="Type a command or search..." />
      <Command.List>
        <Command.Empty>No results found.</Command.Empty>
        <CommandGroups />
      </Command.List>
    </Command.Dialog>
  )
}
```

## Command Groups

Commands are organized into logical groups:

```tsx title="Command groups"
function CommandGroups() {
  return (
    <>
      <Command.Group heading="Navigation">
        <CommandItem icon={<HomeIcon />} onSelect={() => navigate('/')}>
          Dashboard
        </CommandItem>
        <CommandItem icon={<TableIcon />} onSelect={() => navigate('/editor')}>
          Table Editor
        </CommandItem>
        <CommandItem icon={<CodeIcon />} onSelect={() => navigate('/sql')}>
          SQL Editor
        </CommandItem>
      </Command.Group>
      
      <Command.Group heading="Actions">
        <CommandItem icon={<PlusIcon />} onSelect={createProject}>
          New Project
        </CommandItem>
        <CommandItem icon={<TableIcon />} onSelect={createTable}>
          New Table
        </CommandItem>
      </Command.Group>
      
      <Command.Group heading="Recent">
        {recentItems.map((item) => (
          <CommandItem key={item.id} {...item} />
        ))}
      </Command.Group>
    </>
  )
}
```

## Keyboard Navigation

Full keyboard support following standard patterns:

- **Arrow keys**: Navigate between items
- **Enter**: Select item
- **Escape**: Close palette
- **Backspace**: Clear search (when empty, close palette)

## Light Mode Fixes

The initial implementation had color issues in light mode that required fixing:

```css title="Light mode fixes"
[data-theme='light'] [cmdk-root] {
  --cmdk-item-bg: var(--background-surface);
  --cmdk-item-bg-selected: var(--background-overlay);
  --cmdk-border: var(--border-default);
}
```

## Result

The command palette has become a favorite feature for power users, enabling them to navigate the increasingly complex dashboard without hunting through menus.
